### Chrome shortcuts
* `CTRL + N` - new window
* `F11` - full screen
* `F12` - debug tools
* `CTRL + P` - (in debug tools, in **Sources** tab) find a file by name
* `CTRL + F` - (in debug tools, in **Elements** tab) find elements by **XPath** query
